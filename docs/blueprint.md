# **App Name**: HelloWorld

## Core Features:

- Display Greeting: Render a 'Hello World!' message centrally on the single page.

## Style Guidelines:

- Light color scheme evoking a fresh, clear, and inviting feel for a welcoming first impression.
- Primary color: A vibrant yet serene blue, chosen to represent new beginnings and clarity. (RGB: #3A7DE1, HSL: 215, 75%, 55%)
- Background color: An extremely desaturated version of the primary blue, creating a clean and airy base. (RGB: #EDF3F9, HSL: 215, 20%, 95%)
- Accent color: An analogous and slightly darker cyan-turquoise to add a subtle dynamic highlight without being overly bold. (RGB: #2EB1C0, HSL: 185, 70%, 45%)
- Headline and body font: 'Inter' (sans-serif) for its modern, clean, and objective aesthetic, ensuring legibility and a contemporary feel for the 'Hello World' message.
- Centrally align the 'Hello World' message within a clean, minimalist layout, focusing on direct communication.
- A subtle fade-in animation for the 'Hello World' text upon page load, enhancing the sense of welcome and emergence.
