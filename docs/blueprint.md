# **App Name**: Staunch Ventures Platform

## Core Features:

- Homepage Hero Section: Displays a headline, growth partners logo grid, and active initiatives section to capture attention and convey Staunch Ventures' mission.
- About Us Page: Showcases the VC's mission and vision through interactive cards, providing insights into their core values and investment philosophy.
- Initiatives Showcase: Lists current programs with brief descriptions, helping founders discover opportunities for funding and collaboration.
- Contact Form & Office Info: Enables potential founders and partners to reach out via a contact form and displays contact information with office locations for Staunch Ventures.
- Secure Dashboard Layout: Features a persistent sidebar for navigating portfolio, deal room, profile, and community, ensuring investors can quickly access key sections of the platform.
- Financial Dashboard: Presents a mock financial dashboard showing Total NAV, MRR, and Burn Rate using cards and mock charts to give investors an overview of key metrics.
- Deal Room Grid: Displays active startup deals with startup name, sector, and funding ask details, facilitating quick evaluations of potential investments.

## Style Guidelines:

- The site will default to a Dark Mode look. Backgrounds will be dark gray (#334155), which corresponds to slate-800 in Tailwind CSS, text white/off-white.
- Use a vibrant Orange gradient for buttons and key headings. Primary color: Deep Orange (#FF4500).
- Background color: Dark grayish-orange (#36302D). The hue is that of the primary color, desaturated and darkened to function in a dark scheme.
- Accent color: Slightly reddish orange (#FF2A00). It's analogous to the primary color, and carefully tweaked to stand out.
- Clean, sans-serif fonts: 'Inter' for both headlines and body text. This sans-serif is modern and neutral.
- Use `lucide-react` for dashboard icons such as PieChart, Users, FileText, and Settings.
- Cards: Use glassmorphism effects for the Dashboard cards and Initiative grids, applying slightly transparent dark backgrounds with thin borders.
- Use subtle transitions and animations to enhance the user experience when navigating and interacting with elements.
